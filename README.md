# ESP32 Temperature Monitor with OLED 

##  Overview

Real-time temperature and humidity monitoring system built with ESP32, featuring an OLED display and LED indicators.



##  Components Used

* ESP32 Dev Module
* DHT11 Sensor
* OLED Display (I2C)
* LEDs (Red, Yellow, Green)
* Resistors
* Breadboard & jumper wires


##  Features

* Real-time temperature & humidity readings
* OLED display output
* LED status indicators
* Embedded control logic


##  Wiring

| Component  | ESP32 Pin |
| ---------- | --------- |
| DHT11 Data | GPIO 4    |
| OLED SDA   | GPIO 21   |
| OLED SCL   | GPIO 22   |
| Green LED  | GPIO 15   |
| Yellow LED | GPIO 5    |
| Red LED    | GPIO 18   |
